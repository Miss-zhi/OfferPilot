# 前端页面原型

> **来源**：《03-详细设计说明书》§6  
> **模块**：前端页面  
> **技术栈**：Vue 3 + Element Plus  
> **页面数**：9 个核心页面（1 登录 + 4 用户 + 4 知识库管理）
