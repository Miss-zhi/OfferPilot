# 目标公司面试情报增强

## 1. 背景与目标

### 1.1 背景

当前 `CompanySearchTool.search_company_interviews()` 仅返回 `List<CompanyItem>` 片段（公司名、面试类型、摘要、难度分数），子Agent `company_researcher` 靠 LLM 拼凑"面试情报卡"。需求规格说明书（§57-69）要求：

- 结构化"面试情报卡"：高频技术题 Top10、行为面试偏好、技术栈侧重、面试轮次和流程
- 对比用户当前技能水平和目标岗位要求，指出差距

当前实现这些能力完全依赖 LLM 自由发挥，输出质量和一致性无法保证。

### 1.2 目标

1. 新增 `build_interview_card` 工具，输出结构化情报卡（JSON Schema 约束）
2. 新增 `analyze_skill_gap` 工具，对比用户 KnowledgeMastery 与目标岗位要求
3. 增强 `company_researcher` SysPrompt，明确情报卡各字段的生成规范

## 2. 功能清单

| 功能模块 | 功能点 | 优先级 | 说明 |
|---------|--------|--------|------|
| 情报卡 | 结构化面试情报卡生成 | P0 | JSON Schema 约束输出 Top10 高频题/偏好/流程/侧重 |
| 情报卡 | 面试轮次与流程提取 | P1 | 从搜索结果中解析面试轮次信息 |

## 4. 技术方案

### 4.1 整体架构

```
用户输入："字节跳动 后端开发 面试情报"
    ↓
主Agent → spawn company_researcher
    ↓
company_researcher 并行调用：
  ├── search_company_interviews(company="字节跳动", position="后端开发")
  ├── search_questions(keyword="字节跳动 后端 高频题")
  └── web_search(兜底，知识库无结果时)
    ↓
整合搜索结果 → 调用 build_interview_card(搜索结果) → 结构化情报卡
    ↓
(可选) 调用 analyze_skill_gap(userId, company, position) → 技能差距分析
    ↓
输出：面试情报卡 + 个性化备考建议
```

### 4.2 面试情报卡结构化输出（InterviewCardResult）

```java
@Data
public class InterviewCardResult {
    private String companyName;
    private String position;
    private String dataSource;              // "kb" / "web" / "mixed"

    // 高频题 Top10
    private List<FrequentQuestion> topQuestions;

    // 面试风格偏好
    private String interviewStyle;           // "技术驱动" / "行为导向" / "混合型"
    private String behaviorPreference;       // "重视STAR / 关注项目深度 / ..."

    // 技术栈侧重
    private List<String> techStackFocus;

    // 面试流程
    private List<InterviewRound> rounds;

    // 考点分布
    private Map<String, Integer> topicDistribution;  // 主题 → 出现频次

    private String guidance;                 // 供 LLM 整合用的指导文本

    @Data
    public static class FrequentQuestion {
        private String question;
        private String category;
        private Integer frequency;          // 被提及次数
        private String difficulty;
    }

    @Data
    public static class InterviewRound {
        private int roundNumber;
        private String type;                // "技术面" / "HR面" / "主管面" / "笔试"
        private String duration;            // "45分钟" / "1小时"
        private String focus;               // "算法+系统设计" / "项目深挖"
    }
}
```

### 4.3 InterviewCardTool 实现

> **设计说明 — InterviewIntelligenceService 方法边界**：
> - `extractTopQuestions()`：从搜索结果中做词频统计 + 排序，返回原始题目文本片段，**不判断**题目的类别或质量
> - `analyzeStylePreference()` / `analyzeBehaviorPreference()`：基于搜索结果关键词做统计计数（如"STAR"出现次数），**不生成**评语文本
> - `extractTechStackFocus()`：提取搜索结果中高频技术关键词，**不做**"哪些技术栈重要"的判断
> - `parseInterviewRounds()`：解析搜索结果中的结构化轮次信息，返回解析后的原始数据
> - `countTopicDistribution()`：统计搜索结果中各主题出现频次
>
> 所有这些方法的输出都是**结构化统计数据**，不含自然语言评语。评语由 LLM 基于 `buildGuidance` 指导文本生成。

```java
@Slf4j
@Component
@RequiredArgsConstructor
public class InterviewCardTool {

    private final InterviewIntelligenceService intelligenceService;

    @Tool(name = "build_interview_card",
          description = "将搜索结果整合为结构化的面试情报卡，包含高频题、面试风格、流程等信息")
    public InterviewCardResult buildCard(
            @ToolParam(name = "company_name", description = "目标公司名称") String companyName,
            @ToolParam(name = "position", description = "目标岗位") String position,
            @ToolParam(name = "search_results", description = "search_company_interviews 和 search_questions 的拼接结果") String searchResults) {

        log.info("build_interview_card called: company={}, position={}", companyName, position);

        InterviewCardResult card = new InterviewCardResult();
        card.setCompanyName(companyName);
        card.setPosition(position);

        // 1. 解析高频题（按频次排序取 Top10）
        card.setTopQuestions(intelligenceService.extractTopQuestions(searchResults));

        // 2. 分析面试风格偏好（关键词统计）
        card.setInterviewStyle(intelligenceService.analyzeStylePreference(searchResults));
        card.setBehaviorPreference(intelligenceService.analyzeBehaviorPreference(searchResults));

        // 3. 提取技术栈侧重
        card.setTechStackFocus(intelligenceService.extractTechStackFocus(searchResults));

        // 4. 解析面试轮次
        card.setRounds(intelligenceService.parseInterviewRounds(searchResults));

        // 5. 统计考点分布
        card.setTopicDistribution(intelligenceService.countTopicDistribution(searchResults));

        // 6. 构建 LLM 指导文本（不包含成品自然语言）
        card.setGuidance(buildGuidance(card));

        return card;
    }

    private String buildGuidance(InterviewCardResult card) {
        // guidance 必须内联结构化数据，因为 LLM 无法直接访问 Java 对象
        StringBuilder sb = new StringBuilder();
        sb.append(String.format("请根据以下结构化面试情报数据生成一份面试情报卡报告：\n"));
        sb.append(String.format("- 公司：%s，岗位：%s，数据来源：%s\n",
                card.getCompanyName(), card.getPosition(), card.getDataSource()));

        // 内联高频题 Top10
        sb.append("- 高频题 Top").append(card.getTopQuestions().size()).append("：\n");
        for (var q : card.getTopQuestions()) {
            sb.append(String.format("  · %s [%s] 频次=%d 难度=%s\n",
                    q.getQuestion(), q.getCategory(), q.getFrequency(), q.getDifficulty()));
        }

        // 内联面试风格
        sb.append(String.format("- 面试风格：%s，行为偏好：%s\n",
                card.getInterviewStyle(), card.getBehaviorPreference()));

        // 内联技术栈
        sb.append(String.format("- 技术栈侧重：%s\n", card.getTechStackFocus()));

        // 内联面试轮次
        sb.append(String.format("- 面试轮次：%d 轮\n", card.getRounds().size()));
        for (var r : card.getRounds()) {
            sb.append(String.format("  第%d轮 [%s] %s，侧重：%s\n",
                    r.getRoundNumber(), r.getType(), r.getDuration(), r.getFocus()));
        }

        // 内联考点分布
        sb.append("- 考点分布：").append(card.getTopicDistribution()).append("\n");

        sb.append("\n请用专业友好的语气输出完整报告，标注每项信息的数据来源（知识库/互联网）。");
        return sb.toString();
    }
}
```

### 4.4 技能差距分析（SkillGapTool）

```java
@Slf4j
@Component
@RequiredArgsConstructor
public class SkillGapTool {

    private final KnowledgeMasteryRepository masteryRepo;
    private final InterviewIntelligenceService intelligenceService;

    @Tool(name = "analyze_skill_gap",
          description = "对比用户当前技能掌握度与目标公司岗位要求，输出差距分析矩阵")
    public SkillGapResult analyzeGap(
            @ToolParam(name = "user_id", description = "用户ID") String userId,
            @ToolParam(name = "company_name", description = "目标公司") String companyName,
            @ToolParam(name = "position", description = "目标岗位") String position) {

        log.info("analyze_skill_gap called: userId={}, company={}, position={}", userId, companyName, position);

        // 1. 读取用户当前掌握度
        List<KnowledgeMastery> masteries = masteryRepo.findByUserId(userId);
        Map<String, Integer> userScores = masteries.stream()
                .filter(m -> m.getScore() != null)
                .collect(Collectors.toMap(KnowledgeMastery::getKnowledgePoint, KnowledgeMastery::getScore));

        // 2. 提取目标公司岗位的技能要求（从知识库/搜索结果中）
        Map<String, Integer> requiredSkills = intelligenceService.getRequiredSkills(companyName, position);

        // 3. 计算差距矩阵
        List<SkillGapResult.GapItem> gaps = new ArrayList<>();
        for (var entry : requiredSkills.entrySet()) {
            String skill = entry.getKey();
            int required = entry.getValue();     // 目标要求掌握度（0-100）
            int current = userScores.getOrDefault(skill, 0);

            SkillGapResult.GapItem item = new SkillGapResult.GapItem();
            item.setSkill(skill);
            item.setCurrentScore(current);
            item.setRequiredScore(required);
            item.setGap(Math.max(0, required - current));
            // 优先级 = 缺口 × 岗位要求权重（缺口越大且越重要的技能越优先）
            item.setPriority(item.getGap() * required / 100.0);
            gaps.add(item);
        }

        // 按优先级降序
        gaps.sort((a, b) -> Double.compare(b.getPriority(), a.getPriority()));

        String guidance = buildGapGuidance(gaps);
        return new SkillGapResult(gaps, guidance);
    }
}
```

### 4.5 AgentFactory 变更

```java
private SubagentDeclaration buildCompanyAgent() {
    return SubagentDeclaration.builder()
        .name("company_researcher")
        .description("公司面试情报调研员...")
        .inlineAgentsBody("""
            你是面试情报分析师。按以下流程工作：
            1. 调用 search_company_interviews 检索公司面试情报
            2. 调用 search_questions 检索高频考点
            3. 知识库结果不足时调用 web_search 兜底
            4. 调用 build_interview_card 将搜索结果整合为结构化情报卡
            5. 如果用户已登录且有面试记录，调用 analyze_skill_gap 进行技能差距分析
            6. 输出格式：先展示情报卡，再展示差距分析（如有），最后给出备考建议
            情报卡必须标注数据来源（知识库 / 互联网）。""")
        .tools(List.of("search_company_interviews", "search_questions",
            "web_search", "build_interview_card", "analyze_skill_gap"))
        .build();
}
```
