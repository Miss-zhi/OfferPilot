单体 Spring Boot 应用，按包分层组织：
- `controller/` REST 接口层（Auth/Chat/KnowledgeBase/ModelConfig/Progress/Report/Salary/SearchStats/UserModel），仅调用 Service，不直连 Repository。
- `service/` 业务编排层（15+ Service），其中 `ingestion/` 子包封装文档解析→分片→向量化入库流程；`MilvusCollectionManager` 负责动态 Collection 管理。
- `agent/` AgentScope 框架层：`AgentFactory` 通过 Caffeine 有界缓存池（最大 500 个，30 分钟 LRU 淘汰）复用 `HarnessAgent`，按用户维度构建并注册 4 组 11 个 `@Tool` 工具与 7 个 `SubagentDeclaration`（resume_coach/tech_evaluator/expression_evaluator/mock_interviewer/company_researcher/study_planner/salary_advisor），中间件链为 TokenMonitorMiddleware → CostControlMiddleware。
- `entity/` + `repository/` JPA 数据访问层（18 Entity + 15 Repository），配合 `@EnableJpaAuditing` 自动维护审计字段。
- `config/` 集中配置：`SecurityConfig`(JWT 过滤器)、`RedisConfig`/`MilvusConfig`/`AsyncConfig`/`WebConfig`、`AgentScopeProperties` 绑定 `agentscope.*` 配置项。
- 依赖方向严格单向：controller → service → (repository / agent / external SDK)，无反向引用。
- 多环境配置：`application.yml`（公共）+ `application-dev.yml`（H2 + DEBUG）+ `application-prod.yml`（MySQL + INFO），敏感信息通过 `${ENV_VAR:default}` 注入。
