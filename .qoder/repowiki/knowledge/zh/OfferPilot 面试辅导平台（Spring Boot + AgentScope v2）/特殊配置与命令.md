本地开发需先 `docker compose up -d` 启动 Milvus、Redis、MySQL；应用默认走 dev profile（H2 文件库 + H2 Console 在 /h2-console）。运行 `mvn spring-boot:run` 即可，敏感 Key 通过环境变量 `DASHSCOPE_API_KEY`、`EMBEDDING_API_KEY`、`TRANSCRIPTION_API_KEY`、`JWT_SECRET`、`ENCRYPTION_SECRET_KEY` 注入。集成测试 `mvn test` 会由 Testcontainers 自动拉起 MySQL 容器，无需本地安装数据库。
