本模块采用典型的 React 三层拆分：
- UI 层 `ChatPage.tsx`：使用 Ant Design Layout（Sider/Content/Header）构建三栏布局，左侧通过 `CHAT_FUNCTIONS` 常量驱动功能菜单，顶部 Header 显示用户名与清空/退出操作，中间消息区由 `ChatBubble` 组件渲染，底部输入区集成 `FileUploader` 与 TextArea。
- 状态层 `chat-store.ts`：基于 Zustand 的单一 store，维护 `sessionId`、`messages`、`isStreaming`、`activeFunction` 四个字段，并提供 `addMessage` / `appendStreamContent` / `resetSession` 等不可变更新方法；`ChatMessage` 类型限定 role 为 `'user' | 'ai'`。
- 服务层 `chatService.ts`：封装三类后端调用——`sendMessageStream` 使用 `@microsoft/fetch-event-source` 建立 SSE 连接并按 `delta/done/error` 事件分发回调；`sendMessageSync` 作为同步兜底；`uploadFile` 以 multipart/form-data 上传附件。
依赖方向：ChatPage → chat-store + chatService → infra/http + infra/constants，不直接访问 localStorage，token 通过 `STORAGE_KEYS.TOKEN` 读取。
