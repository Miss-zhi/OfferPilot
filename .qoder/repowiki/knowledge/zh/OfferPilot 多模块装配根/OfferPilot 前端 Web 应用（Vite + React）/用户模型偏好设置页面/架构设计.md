单文件 React 函数组件，采用 Ant Design 的 Card + Table + Modal + Form 组合呈现；状态（data / loading / presets / privateModalOpen / form）全部通过 useState/useForm 管理在组件内部，无外部 store 依赖；数据获取通过 `@/service/modelConfigService` 提供的 `getAvailableModels`、`listProviderPresets`、`setDefaultModel`、`createPrivateModel` 四个方法完成，组件仅负责 UI 编排与 message 提示，不直接处理 HTTP 细节；Provider 显示名通过本地 presets 列表按 provider 字段匹配渲染，形成 Provider 预设与用户配置之间的弱耦合映射。
