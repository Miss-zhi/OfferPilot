- Service 层以对象字面量导出（如 `resumeService = { uploadResume: ... }`），方法签名采用箭头函数并直接 `return http.post<T>(...).then(res => res.data)`，避免额外 Promise 包装。
- Zustand store 通过 `create<State>()((set) => ({ state, actions }))` 工厂形式定义，state 初始值与方法并列放在同一对象中，setter 统一通过 `set({ field })` 更新。
- 所有公开导出类型（`ResumeUploadResult`、`KbState`）均使用 `export interface` 显式声明，供组件与服务层共享。
