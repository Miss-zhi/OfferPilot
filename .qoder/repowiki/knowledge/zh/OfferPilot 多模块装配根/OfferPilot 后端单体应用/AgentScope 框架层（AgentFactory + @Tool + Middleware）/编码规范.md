- 每个 Tool 类使用 @Component + @RequiredArgsConstructor 声明为 Spring Bean，方法级用 @Tool(name=, description=) 暴露给 LLM，参数用 @ToolParam(name=, description=) 描述。
- 中间件实现 MiddlewareBase 接口并在每个 hook 中必须调用 next.apply(input) 推进链式执行，统计状态使用 AtomicInteger/AtomicLong 保证线程安全。
- 工具按语义分组注册：先 toolkit.createToolGroup(组名, 描述)，再用 toolkit.registration().tool(...).group(组名).apply() 批量绑定，最后统一 registerMetaTool()。
- Agent 实例按 userId 维度放入 Caffeine 有界缓存，getOrCreateAgent(userId) 复用已存在实例，evictAgent(userId) 强制重建。
