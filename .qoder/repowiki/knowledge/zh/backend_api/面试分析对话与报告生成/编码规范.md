- Controller 方法先通过 `@AuthenticationPrincipal UserDetails` 提取 userId，再交给下游 Service/Agent，不在 Controller 中直接访问 Repository。
- 对外请求体使用 Lombok `@Data` + Jakarta `@NotBlank` 等校验注解定义在独立 `dto.*` 包下，响应统一包裹 `ApiResponse<T>`。
- 业务异常通过构造 `BusinessException(状态码, 消息)` 抛出，由全局 `@RestControllerAdvice` 统一映射到 HTTP 状态码。
- 涉及多表写入的方法（保存问答、生成报告）统一标注 `@Transactional`，并在日志中记录关键 ID 与分数便于追踪。
