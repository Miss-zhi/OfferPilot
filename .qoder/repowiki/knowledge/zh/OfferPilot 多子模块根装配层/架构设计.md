根目录不承载业务代码，仅作为多子模块的装配层：
- `pom.xml` 是 OfferPilot 后端单体（backend_api）的单一构建入口，集中声明 Spring Boot 3.2.5、AgentScope v2、Milvus SDK、JWT、Testcontainers 等版本并配置 Jacoco；frontend_webapp 与 project_docs 各自独立构建，不受此 POM 影响。
- `docker-compose.yml` 定义开发期基础设施容器族（MySQL 8.0、Redis 7、etcd 3.5.14、MinIO、Milvus 2.4.6），应用通过 IDE/Maven 本地启动而非容器化，形成“应用进程 + 外部服务”的解耦部署模式。
- `.env` 提供默认环境变量（数据库密码、MinIO 凭据、DashScope API Key、JWT Secret），被 docker-compose 与后端 application-{dev,prod}.yml 共同消费，实现敏感信息与环境切换的统一注入点。
