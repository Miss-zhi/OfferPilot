本模块由三个独立的 React 函数式组件构成，分别对应三条管理子路径：
- `knowledge/KnowledgeListPage.tsx`：知识库列表页，负责知识库的增删查及分页展示，点击行名通过 `useNavigate('/admin/knowledge/${kbId}')` 跳转到文档详情页。
- `documents/DocListPage.tsx`：单知识库下的文档列表页，支持拖拽上传 PDF/TXT/MD/DOCX、选择分块策略（AUTO/FIXED/SEMANTIC）、批量删除、详情弹窗查看 chunks、以及基于 `setInterval(2000ms)` 轮询 `kbService.getDocProgress` 的异步入库状态机（UPLOADED→PARSING→CHUNKING→EMBEDDING→INDEXING→ACTIVE/FAILED），并在卸载时清理所有定时器。
- `search/SearchTestPage.tsx`：检索调试页，调用 `kbService.search(kbId, {query,topK})` 并以 Tag 颜色区分相似度阈值（<0.3 green / <0.6 orange / else red）。

依赖方向单一向下：页面 → `@/service/kbService`（HTTP 客户端）+ `@/store/kb-store`（Zustand 全局 store，提供 `currentKbName` 与 `setCurrentKb`），不直接访问后端或共享业务逻辑；路由参数 `kbId` 通过 `react-router-dom` 的 `useParams` 注入，形成 `/admin/knowledge/:kbId/documents` 与 `/admin/knowledge/:kbId/search` 的两级导航结构。
