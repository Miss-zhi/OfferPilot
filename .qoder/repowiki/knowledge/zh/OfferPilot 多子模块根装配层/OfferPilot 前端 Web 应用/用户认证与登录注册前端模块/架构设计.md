本模块按职责拆分为三层：
- UI 层 `ui/pages/login/LoginPage.tsx`：基于 Ant Design 的 Card + Tabs 组合表单，同时承载登录与注册两个 Tab，通过 `Form.useForm` 分别维护两组字段并调用 `authService`。
- 服务层 `service/authService.ts`：仅暴露 `login` / `register` 两个方法，统一通过 `@/infra/http` 发起 POST 请求到 `API.AUTH_LOGIN` / `API.AUTH_REGISTER`，并将响应体 `.data` 透传给调用方。
- 状态层 `store/auth-store.ts`：使用 Zustand + `persist` 中间件将 token、userId、username、role 持久化到 localStorage（key 来自 `STORAGE_KEYS`），并提供 `setAuth`、`logout`、`isAuthenticated`（通过 base64 解码 JWT payload 校验 `exp`）三个操作。
依赖方向严格单向：LoginPage → authService → infra/http；LoginPage 通过 `useAuthStore` 订阅状态变更，不直接读写 localStorage。
