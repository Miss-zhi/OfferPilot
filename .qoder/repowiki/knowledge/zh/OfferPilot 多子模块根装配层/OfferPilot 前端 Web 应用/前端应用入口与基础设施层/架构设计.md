模块按职责分为三层：
- `app/`：应用根组件。`main.tsx` 通过 `react-dom/client` 的 `createRoot` 挂载到 `#root`；`App.tsx` 作为唯一路由出口，使用 `react-router-dom` 声明 `/login`、`/chat`、`/reports/:id`、`/resume`、`/admin/*`、`/progress`、`/salary` 等页面路由，所有受保护路由由内部 `AuthGuard` 基于 `@/store/auth-store` 的 `isAuthenticated()` 判断并 `Navigate` 到登录页；同时通过 Ant Design `ConfigProvider` 集中注入 `zh_CN` 语言与 `colorPrimary: #1677ff` 主题。
- `infra/`：横切基础设施。`http.ts` 基于 axios 创建实例（baseURL=`/api`，timeout=60s），请求拦截器从 `localStorage` 读取 `STORAGE_KEYS.TOKEN` 注入 `Authorization: Bearer ...`，响应拦截器根据后端 `ApiResponseBody.code===200` 自动解包 `data`，并在 401 时清除本地状态并重定向 `/login`；`constants.ts` 集中定义所有 REST 路径（`API.*`）、`STORAGE_KEYS` 与 `UPLOAD_LIMITS`，全部以 `as const` 导出供全模块共享。
- `vite-env.d.ts`：仅引用 `vite/client` 类型，使 Vite 资源 import 具备 TS 类型。
依赖方向单向：`app` → `infra` + `@/ui/pages/*` + `@/store/*`，`infra` 不反向依赖业务层。
